云同步
使用 Firebase 免费云端。先本地能用；设置 Firebase 后可以注册账号、上传、下载。
你需要先去 Firebase 建项目：
1. Firebase Console → Add project
2. Build → Authentication → Sign-in method → 开启 Email/Password
3. Build → Firestore Database → Create database
4. Project settings → Web app → 复制 firebaseConfig
5. 粘贴到下面保存
🤖 Claude AI 解析功能：
需要 Anthropic API Key 才能使用 AI解析、生成同类题等功能。
1. 去 console.anthropic.com 注册
2. 左边菜单 API Keys → Create Key
3. 复制 sk-ant-... 开头的 Key 粘贴到下面保存
4. Key 只存在你自己的浏览器里，别人无法使用
